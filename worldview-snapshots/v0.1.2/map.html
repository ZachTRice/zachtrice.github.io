Overview

Worldview Snapshots is a tool for creating a subset of the Global Imagery Browse Services (GIBS) imagery. This tool is a replacement for the Rapid Response Subsets tool.

How To Use

  1. Use the form to select a projection, date, layers, bounding box, and image file options.
  2. Use the preview button to preview and share a subset before downloading.
  3. Use the download button to generate a snapshot.
